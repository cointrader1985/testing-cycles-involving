# testing-cycles-involving
The utilities stored in this repository were assembled over several testing cycles involving transaction execution and service validation. Individual files often address a very specific problem and therefore differ significantly in structure.

Several examples concentrate on improving coverage during execution analysis so that transaction behavior can be reviewed after testing sessions have completed.

Some modules compare locally generated values with external oracles in order to verify consistency between different environments.

A few experiments also investigate API matched workflows where responses from independent services are recorded and compared during development.
